# User Stories

## Roller

- **Admin** — personen som skapar och hanterar önskelistor (t.ex. födelsedagsbarnet)
- **Gäst** — person som besöker en delad önskelista och kan reservera presenter

---

## US-1: Skapa önskelista

**Som** admin
**vill jag** kunna skapa en ny önskelista med namn och beskrivning
**så att** jag kan samla mina önskemål inför ett tillfälle.

## US-2: Dela önskelista via länk

**Som** admin
**vill jag** få en unik delbar URL för min önskelista
**så att** jag enkelt kan skicka den till vänner och familj.

## US-3: Lägga till önskemål

**Som** admin
**vill jag** kunna lägga till önskemål med namn, beskrivning, länk och bild
**så att** gäster vet exakt vad jag önskar mig.

## US-4: Redigera önskemål

**Som** admin
**vill jag** kunna redigera eller ta bort befintliga önskemål
**så att** listan alltid är uppdaterad.

## US-5: Se önskelista som gäst

**Som** gäst
**vill jag** kunna öppna en delad länk och se alla önskemål
**så att** jag kan välja vad jag vill ge i present.

## US-6: Reservera present

**Som** gäst
**vill jag** kunna "paxa" ett önskemål genom att ange mitt namn
**så att** ingen annan köper samma present.

## US-7: Avboka reservation

**Som** gäst
**vill jag** kunna ångra min reservation
**så att** önskemålet blir tillgängligt igen om jag ändrar mig.

## US-8: Se reserveringsstatus (admin)

**Som** admin
**vill jag** kunna se vilka önskemål som är reserverade och av vem
**så att** jag har koll på vad som är "paxat".

## US-9: Dölj reserveringar för andra gäster

**Som** gäst
**vill jag** att andra gäster inte kan se vem som reserverat vad
**så att** det förblir en överraskning vem som ger vad.

## US-10: Hantera flera listor

**Som** admin
**vill jag** kunna skapa och hantera flera önskelistor
**så att** jag kan ha separata listor för t.ex. födelsedag och jul.

## US-11: Admin-inloggning

**Som** admin
**vill jag** kunna logga in för att hantera mina listor
**så att** bara jag kan ändra mina önskelistor.

## US-12: Responsiv design

**Som** gäst eller admin
**vill jag** kunna använda appen på mobil, surfplatta och dator
**så att** jag kan nå listan oavsett enhet.
